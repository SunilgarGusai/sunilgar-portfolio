Dr. Sunilgar L. Gusai Portfolio Website

Files included:
- index.html
- style.css
- script.js
- assets/profile.jpg
- assets/CV_DR_Sunilgar_L_Gusai.pdf

How to view locally:
1. Extract the ZIP file.
2. Open index.html in any browser.

How to host free on GitHub Pages:
1. Create a GitHub account.
2. Create a new public repository named: sunilgar-portfolio
3. Upload index.html, style.css, script.js, and the assets folder.
4. Go to Settings > Pages.
5. Under Branch, select main and /root, then Save.
6. Your website will be live in a few minutes at a GitHub Pages URL.

Before publishing:
- Check LinkedIn URL.
- Add Google Scholar link if you want.
- Replace CV PDF later if you update your CV.
